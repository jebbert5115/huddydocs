Privacy Policy

Last Updated: February 26, 2026

1 Introduction

This Privacy Policy explains how the Game Companion Bot ("the Bot") handles user data. We are committed to protecting your privacy.

2 Data We Collect

✅ Data We Store (Expedition Game Only)

  • Discord User ID – Used as a unique identifier to save your game progress.
  • Game Data – Inventory items, coins, equipped gear, expedition state, and gameplay statistics.
  • Contract Progress – Daily and hourly contract completion tracking.
  • Notification Preferences – Last contract notification date and last update version notified, used to avoid duplicate notifications.
🚫 Data We Do NOT Collect or Store
  • Discord usernames or display names
  • Message content
  • Server information
  • Personal information beyond your Discord User ID
  • Command history outside of active game state
  • Usage analytics

⏱ Data Processed Temporarily

  • When you use a non-game command, Discord sends your interaction to the Bot.
  • The Bot processes the command and responds immediately.
  • No data is stored after the response is sent for non-game commands.

3 Third-Party APIs

The Bot fetches game data from:

  • Metaforge (metaforge.app) – Arc Raiders event data.

These requests do not include any of your personal information. Please review their respective privacy policies for how they handle data.

4 Data Storage

The Bot uses a database to store game progress for the Expedition minigame:

  • Your Discord User ID is stored to identify your game save.
  • Game inventory, coins, and stats are stored persistently.
  • Active expedition state is stored to allow resuming.
  • Contract progress and period data are stored and automatically cleaned up after 7 days.
  • Data is stored securely and is only accessible through Bot commands.

5 Data Sharing

We do not share any user data with third parties.

6 Data Deletion

You may request deletion of your game data by contacting the Bot owner through Discord. All stored data associated with your Discord User ID will be permanently removed.

7 Discord Data

The Bot requires certain Discord permissions to function:

  • Send Messages – To respond to your commands.
  • Use Application Commands – To register slash commands.
  • Read Message Content – To support prefix commands (e.g., arc expedition).

The Bot does not request or use any permissions beyond what is necessary.

8 Children's Privacy

The Bot does not knowingly collect information from children under 13. Use of the Bot is subject to Discord's age requirements.

9 Your Rights

You have the right to:

  • Request access to your stored game data.
  • Request deletion of your stored game data.
  • Stop using the Bot at any time.

10 Changes to This Policy

We may update this Privacy Policy at any time. Changes will be reflected in the "Last Updated" date.
